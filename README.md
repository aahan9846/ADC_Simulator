# Analog-to-Digital Converter (ADC) and Digital-to-Analog Converter (DAC) Simulation

A simulation-based study of the complete **Analog-to-Digital Conversion (ADC)** and **Digital-to-Analog Conversion (DAC)** process. This project demonstrates how continuous analog signals are converted into digital binary data and reconstructed back into analog signals through various signal processing stages.

---

## Overview

Modern digital systems such as microcontrollers, computers, smartphones, and embedded devices cannot directly process analog signals. ADCs convert real-world analog signals into digital data, while DACs perform the reverse operation by converting digital data back into analog form.

This simulation provides a visual representation of each stage involved in both conversion processes, making it suitable for learning and understanding the fundamentals of digital electronics and signal processing.

## ADC Conversion Process

The Analog-to-Digital Converter converts a continuous analog signal into digital binary data through the following stages:

### 1. Analog Input Signal
A continuous analog waveform (typically a sine wave) is used as the input signal.

### 2. Sampling
The analog signal is measured at fixed time intervals to obtain discrete sample values.

### 3. Quantization
Each sampled value is rounded to the nearest available voltage level depending on the ADC resolution.

### 4. Binary Encoding
The quantized voltage levels are converted into binary numbers, producing the digital output.

---

## DAC Conversion Process

The Digital-to-Analog Converter reconstructs the analog signal by reversing the ADC process.

### 1. Digital Binary Input
Binary data generated by the ADC is provided as input.

### 2. Digital Decoding
Each binary number is converted into its corresponding voltage level.

### 3. Staircase Output
The DAC generates discrete voltage levels, resulting in a staircase waveform.

### 4. Reconstruction Filter
A Low-Pass Filter smooths the staircase waveform to reconstruct a continuous analog signal.
